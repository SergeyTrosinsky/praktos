# Популярность как у The Beatles

## О проекте

**Популярность как у The Beatles** — проект по анализу и визуализации данных, посвящённый изучению факторов успеха группы The Beatles. Цель работы — выявить количественные и качественные характеристики творчества коллектива, которые привели к мировой популярности, и сформулировать рекомендации для современных исполнителей.

В рамках проекта обработаны два набора данных:
- дискография The Beatles (285 треков, 45 атрибутов);
- статистика продаж 50 самых успешных музыкальных исполнителей.

На основе анализа построены шесть диаграмм, демонстрирующих связь между популярностью треков, структурой альбомов, тематикой песен, длительностью композиций и коммерческим успехом.

## Технологии

- Python (pandas, matplotlib, seaborn, numpy)
- mplcyberpunk — стилизация графиков в киберпанк-эстетике
- Google Colab — среда выполнения
- Дополнительно: нейросеть ideogram.ai (генерация обложки)

## Данные

1. **The Beatles songs dataset** — содержит информацию о каждом треке: название, альбом, длительность, популярность (условная метрика), позиция в чартах UK, тематика, является ли композиция кавером и др.
2. **Best selling music artists** — включает имена исполнителей, страны, жанры, подтверждённые продажи (сертифицированные единицы) и заявленные продажи. Использован для подтверждения лидерства The Beatles среди всех артистов.

## Визуализации

## Ключевые выводы

На основе анализа данных сформулированы рекомендации для достижения уровня популярности The Beatles:

1. **Начинать с каверов** — на раннем этапе карьеры исполнение известных песен помогает собрать минимальную аудиторию.
2. **Выбрать жанр рок** — именно рок-композиции доминируют в дискографии группы и наиболее востребованы публикой.
3. **Выпускать альбомы объёмом не менее 24 песен** — анализ показывает, что наиболее успешные альбомы The Beatles содержат большое количество треков.
4. **Основная тематика — любовь** — более 60% песен группы посвящены любовной лирике.
5. **Длительность трека — не менее 3 минут** — существует положительная корреляция между длительностью (в диапазоне 2–4 минут) и уровнем популярности.
